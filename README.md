# Simple CRM

Simple CRM is a lightweight Customer Relationship Management system designed to manage customer information, track interactions, and schedule follow-ups efficiently.
The application is built using React and Firebase.

## How to Run
1. npm install
2. npm run dev

## Features
- Login / Signup
- Add customer
- Firebase backend

Replace Firebase keys in src/firebase.js
